This is the mern stack project using gemini api.

Frontend: React.js (with hooks, context API, and modern UI components)

Backend: Node.js + Express.js (RESTful API architecture)

Database: MongoDB (Mongoose ODM for schema modeling)

AI Integration: Gemini API (Google’s generative AI platform)

Authentication: JWT‑based authentication & authorization

Deployment: Configurable for cloud platform vercel
